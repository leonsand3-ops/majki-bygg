# Audio Assets

Place audio files here. The game references the following files:

## Music
- `music_menu.ogg` - Main menu background music
- `music_arena.ogg` - In-game arena music

## SFX
- `engine.wav` - Vehicle engine loop
- `impact.wav` - Vehicle collision
- `head_hit.wav` - Kill hit sound
- `explosion.wav` - Explosion
- `button_click.wav` - UI button click
- `victory.wav` - Victory fanfare
- `countdown.wav` - Countdown beep
- `round_start.wav` - Round start signal

All audio files are optional — the game will run silently if they are missing.
Use royalty-free audio from sources like freesound.org or opengameart.org.
